# 🎮 Hangman Game

A simple console-based Hangman game built using Python.

## 📌 Features

- 🎲 Random word selection
- 🔤 Letter-by-letter guessing
- ❌ Maximum 6 incorrect guesses
- ✅ Input validation
- 🏆 Win/Lose conditions
- 💻 Console-based gameplay

## 🛠️ Technologies Used

- Python 3
- VS Code (or any Python IDE)

---

## 📚 Python Concepts Used

- Random Module
- Lists
- Strings
- Loops (while)
- Conditional Statements (if-else)
- Functions
- User Input

---

## ▶️ How to Run
```
python hangman.py
```
## 🎮 How to Play

1. The computer randomly selects a word.
2. The word is displayed as underscores (_).
3. Guess one letter at a time.
4. If the guess is correct, the letter is revealed.
5. If the guess is wrong, a part of the Hangman is drawn.
6. You have *6 incorrect guesses*.
7. Guess the complete word before the Hangman is completed.
